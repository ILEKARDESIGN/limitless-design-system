Interval, Uncertainty
& Statistical
When precision, confidence, tolerance, or risk bands matter — encoding uncertainty directly into the visualization.
All Interval & Statistical Types
Error bar chart · error band chart · confidence interval chart · prediction interval chart · candlestick chart · OHLC chart · range bar chart · floating range bar · high-low chart · min-max band · fan chart · control chart · funnel plot · forest plot · coefficient plot · caterpillar plot · meta-analysis interval plot · survival curve · Kaplan–Meier curve
Error Bar / Confidence Interval / Prediction Band
Best for → Statistical uncertainty · Research results · Model confidence
Overlays interval markers on data points to show measurement uncertainty or prediction ranges. Cyan I-beam error bars with pink dot point estimates and a 10% pink confidence band fill. Covers error bar chart, error band chart, confidence interval chart, prediction interval chart, min-max band, and fan chart.
2026 Styling Direction
Cyan I-beam whiskers · Pink point estimate dots · 10% opacity confidence polygon · 2px stroke line through estimates · Dark plot area · No chart chrome clutter
Clinical Trials ML Model Output Financial Forecasts
Candlestick / OHLC / Range Bar
Best for → Financial data · Price ranges · Open-High-Low-Close
Shows four price points per period — open, high, low, close — as a body + wick formation. Yellow bodies for bullish (up), pink for bearish (down). Covers candlestick, OHLC chart, range bar chart, floating range bar, and high-low chart.
Stock Analysis Crypto Trading Commodity Prices
Forest Plot / Coefficient / Caterpillar / Meta-Analysis
Best for → Meta-analysis · Effect sizes · Research synthesis
Horizontal confidence intervals with point estimates for multiple studies, anchored to a null-effect reference line. Pink diamond point estimates, cyan interval whiskers, yellow pooled summary diamond. Covers forest plot, coefficient plot, caterpillar plot, meta-analysis interval plot, funnel plot, and control chart.
Systematic Reviews Regression Coefficients Quality Control
Survival Curve / Kaplan–Meier
Best for → Time-to-event analysis · Clinical outcomes · Retention analysis
Step-function showing the probability of survival (or retention) over time, with censoring tick marks for incomplete observations. Pink solid for treatment group, cyan dashed for control. Also relevant for SaaS retention cohort analysis beyond clinical use.

Group 08 · Suite 2
Flow, Process
& Journey
Movement, paths, conversion, drop-off, transfer, journeys, and system flows.
All Flow Types
Sankey diagram · alluvial diagram · chord diagram · arc diagram · flow map · path map · user journey flow · process flow · decision tree flow · conversion funnel · stepped funnel · retention funnel · pipeline chart · stage progression chart · migration flow diagram · handoff flow · source-to-destination flow · traffic flow diagram · stream of events chart
Sankey / Alluvial / Flow Diagram
Best for → Flow quantities between stages · Source-to-destination · Budget allocation paths
Width-encoded flow paths between node columns showing how quantities move through a system. Covers Sankey diagram, alluvial diagram, migration flow, source-to-destination flow, traffic flow, handoff flow, and stream of events. Node colours from the neon palette, flow bands at 10–12% opacity for layering without occlusion.
2026 Styling Direction
Rounded node rectangles (3px radius) · Curved Bézier flow paths · 10–12% opacity flow bands · Neon palette node colouring · Barlow Condensed 900 node labels · Dark background for maximum flow contrast · Width encodes quantity

Conversion Funnel / Stepped Funnel / Pipeline
Best for → Conversion drop-off · Sales pipeline · Onboarding stages
Progressively narrowing trapezoids showing volume at each stage of a process. Covers conversion funnel, stepped funnel, retention funnel, pipeline chart, and stage progression chart. Each stage uses a different neon colour with Bebas Neue centre counts.

Chord Diagram / Arc Diagram
Best for → Inter-group relationships · Migration between categories · Bi-directional flows
Circular layout with arcs connecting groups and internal chords showing flow volume between them. Covers chord diagram and arc diagram. Outer arcs use distinct neon colours, internal chords at 15–25% opacity with width encoding volume. Covers decision tree flow and user journey flow as structural process variants.

Nested structures, orgs, taxonomy, budgets, rollups — parent-child relationships in data.
All Hierarchy Types
Tree diagram · node-link tree · dendrogram · radial tree · cluster dendrogram · icicle chart · treemap · sunburst · circle packing · partition chart · org chart · genealogy chart · mind-map style tree · collapsible tree · indented tree table
Tree Diagram / Org Chart / Dendrogram / Node-Link Tree
Best for → Organisational hierarchy · Category taxonomy · NDIS plan structure
Node-link structures showing parent-child relationships across levels. Covers tree diagram, node-link tree, dendrogram, radial tree, cluster dendrogram, org chart, genealogy chart, mind-map style tree, collapsible tree, and indented tree table. Root nodes in pink, branches in neon palette, leaf nodes with subtle stroke + fill treatment. Icicle chart, treemap, sunburst, circle packing, and partition chart are covered in Group 04 as part-to-whole variants.
2026 Styling Direction
Rounded node rectangles (3–4px radius) · Neon fills for parent nodes · Stroke-only leaf nodes with 15% opacity fills · 1px connector lines at 10–15% opacity · Barlow Condensed labels in nodes · JetBrains Mono leaf codes · Top-down vertical layout default

Group 10 · Suite 2
Network
& Graph
Relationships between people, entities, systems, dependencies, and communication pathways.
All Network Types
Force-directed graph · network diagram · node-link diagram · adjacency matrix · bipartite graph · dependency graph · knowledge graph view · social network graph · mesh / topology map · radial network · clustered network · weighted network · edge-bundled graph · graph timeline hybrid
Force-Directed Graph / Network / Node-Link
Best for → Relationship mapping · Dependency visualization · Social networks
Nodes positioned by simulated forces with edges showing connections. Node size encodes importance, edge width encodes relationship strength. Covers force-directed graph, network diagram, node-link diagram, social network graph, dependency graph, knowledge graph view, mesh/topology map, radial network, clustered network, weighted network, edge-bundled graph, and graph timeline hybrid. Also covers adjacency matrix and bipartite graph as alternative encodings. Central node (pink) with neon satellite nodes, edges at 10–20% opacity.
2026 Styling Direction
Solid neon node fills · Bebas Neue node labels · Edge opacity 10–20% with width = weight · Size hierarchy: hub nodes 12–14px, leaf 6–8px · Dark background for maximum node contrast · Physics-based layout for organic feel · Pink for central/highest-degree nodes

Choropleth / Region / Filled Map / Cartogram
Best for → Regional density · Geographic comparison · Population data
Fills geographic regions with colour intensity proportional to a data value. Covers choropleth map, filled map, region map, cartogram, and tile grid map. Uses pink at opacity levels from 20% (low) to 90% (high) on a dark background with black strokes between regions.

Bubble Map / Symbol Map / Pin / Cluster / Dot Density
Best for → Point-located data · Magnitude at locations · Service provider mapping
Places sized symbols at geographic coordinates. Covers symbol map, bubble map, proportional symbol map, dot density map, pin map, cluster map, and geofence map. Each location gets a neon-coloured bubble with 20–30% opacity halo and solid 3px centre dot. Size encodes magnitude.

Risk Matrix / Priority Matrix / Impact-Effort
Best for → Risk assessment · Prioritization · Decision frameworks
A 2×2, 3×3, or 5×5 grid where cell colour encodes severity, priority, or urgency. Colour ramps from low (white/cyan at 15%) through medium (yellow at 35%) to high (orange at 60%) to critical (pink at 85–100%). Covers risk matrix, priority matrix, impact-effort matrix, and confusion matrix.
2026 Styling Direction
2px radius cells · Colour ramp: transparent → cyan → yellow → orange → pink · 3px gap between cells · Dark background · JetBrains Mono axis labels · No borders — colour does all the work

Retention Cohort / Feature Adoption / Activity Matrix
Best for → Cohort analysis · Retention tracking · Engagement over time
Triangular grid where rows are cohorts (by signup month) and columns are periods since signup, with cell opacity encoding retention rate. 100% = full pink, diminishing with churn. Covers retention cohort heatmap, cohort matrix, feature adoption matrix, day/hour activity matrix, contribution heatmap, and clustered heatmap. Also covers data matrix, adjacency matrix, cell heatmap, table heatmap, text-table heatmap, calendar heatmap (Group 03), and highlight table.
