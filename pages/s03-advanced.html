Gantt Chart / Resource Timeline / Dependency Timeline
Best for → Project scheduling · Task dependencies · Timeline planning
Horizontal bars on a time axis showing task duration, dependencies, and milestones. Each task uses a different neon colour, with a pink dashed "today" marker. Covers Gantt chart, resource timeline, dependency timeline, release roadmap, milestone timeline, and program board. Yellow diamond markers for milestones, arrow connectors for dependencies.

Best for → Financial decomposition · Profit breakdown · Budget variance
Sequential waterfall showing how revenue flows to net income through cost deductions and additions. Cyan for totals and positive items, pink for deductions, yellow for subtotals, orange for variable costs. Best for → Portfolio performance · Capital trajectory · Runway estimation
Cumulative value line showing portfolio or cash balance over time with drawdown zones highlighted. Yellow for the equity/balance curve, pink shaded areas for drawdown periods below the high watermark, dashed pink high watermark reference. Pictogram / Icon Array / People Chart / Isotype
Best for → Accessible proportion display · Public-facing reports · Non-technical audiences
Uses repeating human-shaped or object icons where filled = counted and outline = remaining. Covers pictogram chart, icon array, isotype chart, people chart, person-icon chart, inventory block chart, seat map fill, occupancy icon grid, and product box fill display. Radial Bar / Circular Bar / Multi-Ring Progress
Best for → Multi-category completion · Apple Watch-style activity rings · Compact KPIs
Concentric rings where arc length encodes value. Each ring is a different category with its own neon colour, creating an Apple Watch Activity-style display. Covers radial bar chart, circular bar chart, radial column chart, multi-ring donut, and circular progress ring. The LIMITLESS infinity symbol anchors the centre. Track rings at 4% opacity, data arcs with round linecaps. Donut, pie, nested donut are in Group 04; radar/spider in Group 06; chord/arc in Group 08; sunburst/radial tree in Group 04/09.
2026 Styling Direction
10px stroke-width rings · 13px gap between rings · 4% opacity track circles · Round linecap arcs · stroke-dasharray animation for reveals · Pink/Cyan/Yellow/Orange ring order · Bebas Neue or ∞ centre label · Dark background for neon ring contrast

Polar Area / Rose / Coxcomb / Nightingale
Best for → Cyclical comparison · Seasonal patterns · Wind/directional data
Wedge segments where radius (not angle) encodes magnitude. Equal angular slices with varying radius create the characteristic rose shape. Covers polar area chart, rose chart, coxcomb chart, and nightingale rose chart. Also covers radial line chart, radial area chart, radial heatmap, and circular timeline. Each wedge uses a neon palette colour at 70% opacity.

Isometric Bar / 3D Column / Perspective Treemap
Best for → Marketing presentations · Executive reports · Visual impact
Isometric projection creates a pseudo-3D effect using three-face polygon rendering. Left face darker, right face medium, top face brightest — all derived from the same neon colour at different opacities. Covers 3D bar chart, 3D column chart, isometric chart, perspective treemap, and pseudo-3D donut. Use deliberately — 2D is almost always clearer for analysis.
2026 Styling Direction
Three-polygon per bar: left 80%, right 60%, top 100% opacity · Base parallelogram at 30% · Neon palette per bar · No actual 3D rendering — SVG polygon isometry · Dark background · Use sparingly for hero visuals, not data analysis

3D Scatter / Surface / Globe / Terrain / Volumetric
Best for → Three-variable relationships · Scientific visualization · Spatial data
Points suspended in a 3D coordinate system with floor shadow projections showing their 2D position. Dashed vertical drop lines connect each point to its shadow. Covers 3D scatter plot, 3D bubble plot, 3D surface plot, 3D heatmap surface, terrain/data-topography view, volumetric density plot, extruded map, 3D network, and globe visualization. Each point uses a neon colour with floor shadows at 10% opacity.
