Calendar Month View
Best for → Overview of month's commitments · Deadline awareness · Scheduling gaps
7-column grid with day numbers, colour-coded event pills, today highlight (pink border + tint), and dimmed adjacent-month days. Each event uses a neon colour by type: pink = SC meetings, cyan = assessments, yellow = deadlines, orange = renewals, purple = referrals. Compact 7px Barlow Condensed event labels with text-overflow ellipsis.

Best for → Today's appointments · Time blocking · Billable hour tracking
Vertical time-slot grid with coloured event blocks positioned at their scheduled time. Block height corresponds to duration. Each block shows participant name, type, location, and billable hours in JetBrains Mono. Colour by activity type. Empty slots visible as grey tracks for gap identification.

Task List / Checklist with Priority & Due Dates
Best for → Action item tracking · Follow-up management · Accountability
Vertical list with checkbox toggles, task names, due dates, priority indicators, and completion state. Completed tasks show pink filled checkbox with strikethrough. Overdue tasks flag in orange with "OVERDUE" label. Left border colour indicates priority tier: pink = high/overdue, orange = urgent, cyan = standard. JetBrains Mono metadata line shows due date and priority.
Design Specifications
16px checkbox with 4px radius · Pink fill + white ✓ when done · 3px left border accent by priority · Strikethrough + 40% opacity for completed · JetBrains Mono 7px due/priority meta · Dark card rows with 3% white background · Sortable by due date, priority, status

Taskboard / Kanban Board
Best for → Workflow status at a glance · Drag-and-drop task management · Team visibility
Three-column board (To Do / In Progress / Done) with draggable task cards. Each card has a left-border priority accent, participant name, due date, and status chips. Column headers show counts in neon-coloured badges. Done column cards fade to 50% opacity. Overdue cards get pink HIGH chip + orange OVERDUE chip. Cards are filterable by participant, priority, and due date.

Project List with Subtasks (Expandable)
Best for → Case management · Multi-step workflows · Progress tracking with detail
Collapsible project rows with progress bar, percentage, due date, and expandable subtask lists. Each project uses a neon accent colour. Expanded projects show subtask checklist with individual due dates, overdue flags, and strikethrough for completed items. The progress bar fills proportionally based on subtask completion. ▾ = expanded, ▸ = collapsed. This is the ClickUp/Asana list view adapted for SC case management.
Design Specifications
Project row: 4% white background · Barlow Condensed 900 12px project name · JetBrains Mono 7px subtask count + due date · 60px progress bar with neon fill · Bebas Neue 14px percentage · Subtask rows: 12px checkbox · 9px task text · JetBrains Mono 6px due dates · Pink done checkboxes · Orange OVERDUE flags · Collapsible with ▾/▸ toggle

Project Board with Subtasks (Kanban-Style)
Best for → Multi-project overview · Case status board · Portfolio management
Kanban columns (Not Started / Active / Complete) where each card represents a project/case with embedded progress bar, subtask count, and expandable subtask preview. Active cards show remaining subtasks inline, with overdue items flagged in orange. Each card's left border and progress bar use the project's assigned neon colour. This is the Motion/ClickUp board view adapted for SC caseload management — each "project" is a participant case with discrete action items as subtasks.
